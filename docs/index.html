Project Overview

A comprehensive email marketing automation platform powered by artificial intelligence

What is this project?

This is a full-stack email marketing automation platform that leverages AI to create personalized email campaigns. Built with modern technologies and designed for scalability and commercial use.

  • MIT Licensed for commercial use
  • Monorepo architecture with Turborepo
  • Next.js frontend with NestJS backend
  • AI-powered content generation
  • Real-time analytics and reporting
